-------------------- Bicycle Renting APP  -----------------------------

La aplicacion esta creada con Vite v4 con REACT v18.

Todas las librerias utilizadas , arquitectura de carpetas , implementacion de logica , manejo de estado con CONTEX API, configuracion de libreria de estilos , Router , entre otras 
se han implementado con la finalidad de hacer la aplicacion escalable.

El estilo de la aplicacion es basico , amigable a la vista , con claras  mejoras por implementar.

Informacion del proyecto :

- Configuracion de Router con react-router-dom v6
- Configuracion personalizada con libreria de estilos MUI (MATERIAL UI)
- Implementacion y configuracion del manejo global del estado de la aplicacion con CONTEX API.
- Utilizacion de libreria dayjs para manejo de fechas.
- Implementacion de libreria yup para validacion de formulario para la reseva.
- Implementacion de interfaz de usuarion para la seleccion de fecha con libreia externa react-day-picker.
- Proyecto configurado para futura implementacion de tipado con typescript.


Como Ejecutar la aplicacion:

- obtener los archivos del proyecto desde repositorio de git  
- instalar todas las dependecias del proyecto con el comando ---->  npm install
- Ejecutar aplicacion con el comando --------> npm run dev.
